\section{基本规则}

姊姊好！
行文的基本规则和 Markdown 其实差不太多：
一段话内容由若干行连续的文字组成。
两段话之间由至少一个空行分隔开来。
注意你在源文件（\path{xxx.tex}）
里加入多少个空行都会被当成一个空行来对待，
这一点对于空格来说也是一样的。
使用 \LaTeX\ 时建议先不要管排版，
所以不要想着要在最后的文档中哪里插入空行或者空格。
你只要专注于你想要写的「内容」上，
而不用担心「格式」上的问题。
